# House Price Predictor

## Overview
This project predicts house prices based on area and number of bedrooms using a Linear Regression model.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Accepts multiple inputs
- Predicts house prices
- Demonstrates regression with multiple variables

## Example Output
Enter area: 1450  
Enter bedrooms: 2  
Estimated Price: 290000.00

- Training regression models
- Making predictions with real-world data
